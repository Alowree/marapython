CSS 的文字属性主要有 `font-style`，`font-weight`，`font-size` 和 `font-family` 这四种。本文将主要研究 `font-family` 相关知识要点。

> The **`font-family`** CSS property specifies a prioritized list of one or more font family names and/or generic family names for the selected element.
>
> CSS 属性 **`font-family`** 允许您通过给定一个有先后顺序的，由特定字体名称或者通用字体名称组成的列表来为选定的元素设置字体。

通常，我们会选择给正文文本配置一系列的「特定字体 + 通用字体」，给代码文本配置另外的「特定字体 + 通用字体」。

## Syntax

```
CSS
element {
font-family: <family-name> | <generic-family>
}

.class {
font-family: <family-name> | <generic-family>
}

#id {
font-family: <family-name> | <generic-family>
}
```

## 特定字体名称

a specific font family name

## 通用字体名称

a generic font family name

中文字体

英文字体

本地字休

网络字体
