Since the release of HTML5, audios can be added to webpages using the <code>audio</code> tag. Previously, audios could be only played on web pages using web plugins like Flash.

```markdown
<figure>
    <figcaption>Abracadabra</figcaption>
    <audio
        controls
        src="../../audio/Abracadabra.mp3">
            Your browser does not support the
            <code>audio</code> element.
    </audio>
</figure>
```
