## 定义

> In Python, **literals** refer to the data that is specified in a variable or constant. So, literal basically corresponds to a short and easy way to specify a value.

在 Python 中，**字面量**是指在变量或常量中指定的数据。 因此，字面量基本上就是一种指定数值的简短方法。

> “A constant can be defined as an entity that has a fixed value or an entity whose value does not vary.”
> Well, we luckily have the concept of constants in Python as well! They are known as literals in Python.
> **Python literals** are quantities / notations whose value does not change during the execution of a program.

“常数可以定义为具有固定值的实体或值不变的实体。”
好吧，幸运的是，我们在 Python 中也有常量的概念！ 它们在 Python 中被称为字面量。
**Python 字面量**是数量的表述，其值在程序执行期间不会改变。

> Generally, **literals** are a notation for representing a fixed value in source code. They can also be defined as raw value or data given in variables or constants.

通常，**字面量**是用于在源代码中表示固定值的符号。 它们也可以定义为变量或常量中给出的原始值或数据。

> **Python Literals** can be defined as data that is given in a variable or constant.

**Python 字面量**可以定义为在变量或常量中给出的数据。

> **Literals** are the constant values or the variable values used in a Python code.

**字面量**是 Python 代码中使用的常量值或变量值。

## 五种类型

### String Literals

- string literals
  - single-line string literal
  - multi-line string literal
- character literal
- empty string literal or empty character literal

### Numeric Literals

- int literals
- float literals
- complex number literals

### Boolean Literals

Boolean literals in Python are pretty straight-forward and have only two values-

- True - True represents the value 1
- False - False represents the value 0

### Special Literals

- Python literals have one special literal which is known as “None”. This literal in Python is used to signify that a particular field is not created.

  When we print the variable that has no value assigned to it, Python will print “None” as output. None is also used for end of lists in Python.

### Collection Literals

- list literals
- tuple literals
- dict literals
- set literals
