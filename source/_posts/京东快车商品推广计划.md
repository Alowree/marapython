## 什么是京东快车

**京东站内搜索流量**，按 CPC/CPM 的付费方式实时竞价，通过**关键词**出价，将目标人群和潜在用户一网打尽。

京东快车是精准的『人找货』付费推广模式。

## 推广类型

- 普通计划：商品推广、腰带店铺推广（品牌调性没那么强的店铺）、活动推广、门店商品推广
- 京选店铺计划：品牌调性很强的店铺可以考虑去投

建议使用：【商品推广】

## 圈定哪些人群

策略：避免跟头部品牌商家例如 BOSE、JBL、哈曼等直接竞争行业大词，针对某个小众目标群体定向投放。

例如：精致的妈妈群体、时尚的城市白领、入睡困难的人群、其他人群。。。？

### 关键词分析

1、找词和筛词

#### 方法一

```
京准通首页 >> 数据中心 >> 行业分析 >> 关键词分析 >> 关键词解析
```

可查看备选关键词的趋势分析，包括展现指数、点击指数、平均出价、点击率、平均点击成本、千次展示成本、转化率和 ROI 等指标。

#### 方法二

```
京东快车 >> 推广管理 >> 普通计划 >> 所有推广计划 >> 推广计划 >> 推广单元 >> 关键词-普通计划 >> 添加关键词
```
