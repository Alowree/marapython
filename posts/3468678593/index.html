Hexo Butterfly使用指南

发表于2021-09-12|更新于2021-09-12|Hexo
|字数总计:5.3k|阅读时长:31分钟|阅读量:

Configuration

  • os: Windows_NT 10.0.19042 win32 x 64
  • node: 14.16.1
  • hexo: 5.4.0
  • butterfly: 3.8.3

This section comes directly from User Guide Butterfly 安裝文檔(二) 主題頁面

Front-matter

Front-matter is the top area of each markdown file, used to designate various variables of certain files.

  • Page Front-matter is used for Page configurations
  • Post Front-matter is used for Post configurations

Tags Page

Remember to add type: tags into the Front-matter of Tags Page.

Categories Page

Remember to add type: categories into the Front-matter of Categories Page.

Link Page

Here are the steps to create a Link Page for your blog.

Create a Link Page

  1. Go to root directory
  2. Input hexo new page link
  3. source/link/index.md will be generated
  4. Add type: link into the Front-matter of index.md

Add Links

Create a link.yml file under path source/_data

class_name and class_desc both support html tags. This part of content as in source/_data/link.yml will be displayed on the top of your link Page, with layout preset by Butterfly theme.

Customize the layout

If you don’t like the default layout and want to customize for your own, do it in source/link/index.md file.

Gallery

Gallery Group Page

Just an ordinary page, so use hexo new page xxxx to create a page.

To display Gallery Group in your page, just directly use Tag Plugin galleryGroup in your Typora editor.

For clarity purpose, use quotation marks in case any contents in title, description, path, or image are too long, although such quotation marks can be omitted.

Gallery Sub Page

Sub Page is also an ordinary page, so use hexo new page xxxx to create a page.
To display Gallery in your page, just directly use Tag Plugin gallery in Typora editor.

Image links (to be added after fixing Gitee image server)

If you want to use a /gallery/chicago link format to display your photo content, you could simply move the entire chicago folder into the gallery folder.

Tag Plugins

Tag Plugins comes along with Hexo, but not a standard Markdown, so you won’t get a actual preview in your Typora editor and you will have to stroke hexo cl, hexo g, and hexo s to have a look before deploy.

These use examples are valid for Butterfly Only.
