Since the release of HTML5, audios can be added to webpages using the audio tag. Previously, audios could be only played on web pages using web plugins like Flash.


A file extension or file name extension is the ending of a file that helps identify the type of file in operating systems. In Microsoft Windows, the file name extension is a period that is often followed by three characters but may also be one, two, or four characters long.

Make sure that the file extension implemented in your code should be consistent with that of your actual music files. For instance, if your implementation uses .mp3 while your actual file uses .MP3 instead, then the audio element will not work properly.
