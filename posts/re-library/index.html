Python内置模块之re库

发表于2021-12-26|更新于2022-01-18|Python
|字数总计:383|阅读时长:1分钟

Python中常用的关于正则表达式的函数是re.search()，使用它必须首先导入正则表达式对应的包（package），也就是输入下面的代码。

通常的用法是提供两个参数：re.search(pattern,string)，其中pattern是字符串形式提供的正则表达式，string是需要匹配的字符串；如果能匹配，则返回一个re.Match object（详细介绍请参考第254页，暂时可以不必关心），这时提示符会显示类似<re.Match object; span=(0, 1), match='4'>之类的结果；如果不能匹配，结果是None（这是Python中的一个特殊值，类似其他某些语言中的Null），不会有任何显示。图1-3演示了运行Python语句的结果。

图1-3 观察re.search()匹配的返回值

span(0, 1)或者spand(3, 4)表示已匹配文本在原字符串中的位置，它的表示方法跟字符串切片的表示方法s[0:1]或者s[3:4]有些类似。

In the above examples, the return value is always the leftmost possible match. re.search() scans the search string from left to right, and as soon as it locates a match for <regex>, it stops scanning and returns the match.
